Welcome to my official development log for **CERN-HSF**. This project focuses on porting the **SOFIE** inference engine to a portable GPU backend using **alpaka**.

* **Phase:** Application Review / Community Bonding
* **Key Goal:** Addressing the 4 critical GPU pipeline bugs identified during the evaluation task.
* **Next Milestone:** Implementing parallel reduction kernels for `ReduceSum` and `ReduceMean`.
